## 알려진 패턴 (오탐 아닌 것)
- "SHA xxx already verified, skipping" → deploy-verify 정상 동작, 에러 아님
- "No fixable issues found" → auto-fix 정상, 에러 아님
- "data-healer complete — all healthy" → 정상
- "Cannot reach jepo@172.30.1.16" → healthcheck 자기 자신 SSH 오탐 (cron 네트워크 타이밍)
- "Claude call limit reached" → API 속도 제한, 자동 다음 실행에서 재시도

## 알려진 실제 에러 패턴
- "Operation not permitted" → macOS 보안 정책 (Full Disk Access 필요)
- "syntax error near unexpected token" → bash 스크립트 문법 오류 (heredoc/$ 이스케이프 확인)
- "non-fast-forward" → git push 충돌 (pull --rebase 필요)
- uvicorn 메모리 급증 → indicator_cache 무한 성장 (재시작 필요)
- "SIGTERM" / exit -15 → 프로세스 종료 신호 (의도적일 수 있음)
